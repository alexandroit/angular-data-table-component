@revivejs/angular-data-table-component

A community-style Angular data table component with sorting, row selection, expandable content, client or server pagination, conditional styles, and a demo layout that tracks the Angular release line.

Declarative columns Keep column definitions in TypeScript and let Angular templates handle the rich cells.
Native Angular patterns Custom cells and expanded content use TemplateRef, not framework adapters.
Stateful table UX Sorting, pagination, selection, and expandable rows are available out of the box.
Versioned docs This repository follows the Angular major line with a dedicated docs build per version.

Quick API reference

Component inputs

Input Description
columnsDeclarative column definitions with selectors, sorting, and templates.
dataRow data to render.
selectableRowsEnables row selection.
expandableRowsEnables expandable detail rows.
expandableRowTemplateAngular template used for expanded content.
paginationTurns on built-in pagination UI.
paginationServerLets the container control which rows are currently loaded.
conditionalRowStylesApplies row styles when predicates match.
themeSwitch between the default and dark themes.

Column capabilities

Field Description
selectorRead values with a row function or a property key.
formatReturn custom display values before rendering.
cellTemplateUse an Angular template for richer cell layouts.
sortableTurns on built-in sorting for that column.
sortFunctionOverride the comparison logic for a specific column.

Outputs

Output Description
sortChangeFires with the current column, direction, and visible rows.
selectedRowsChangeReturns the selection state and selected rows array.
pageChangeFires when the current page changes.
rowsPerPageChangeFires when the footer page size changes.
rowClickedFires with the clicked row.
rowExpandToggledFires when an expandable row opens or closes.
